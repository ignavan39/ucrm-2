## Installation

```bash
$ yarn install
$ cp .env.example .env
```

## Running the app

```bash
$ docker compose up postgres

# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test
